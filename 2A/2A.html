2A - ¿Cuántos Elementos Distintos?

Tiempo límite: 2 segundos

Memoria límite: 256 MB

Dado un arreglo aa de nn elementos:

Sea f(l,r)f(l, r) la cantidad de elementos distintos de al,al+1,⋯,ara_l, a_{l+1}, \cdots, a_r.

Calcule ∑l=1n∑r=lnf(l,r)\sum\limits_{l=1}^{n} \sum\limits_{r=l}^{n} f(l,r)

Es decir, la suma de f(l,r)f(l, r) para todo par (l,r)(l, r) tal que 1≤l≤r≤n1 \leq l \leq r \leq n.

Entrada:

La primera línea contendrá un entero nn (1≤n≤200000)(1 \leq n \leq 200\,000), la cantidad de elementos de arreglo aa.

La segunda línea contendrá nn elementos a1,a2,⋯ana_1, a_2, \cdots a_n (1≤ai≤n)(1 \leq a_i \leq n), los elementos del arreglo aa.

Salida:

En la primera línea, imprima ∑l=1n∑r=lnf(l,r)\sum\limits_{l=1}^{n} \sum\limits_{r=l}^{n} f(l,r)

Subtareas:

  • Subtarea 1: n≤100n \leq 100, todos los elementos de aa son 11, es decir, ai=1a_i = 1 para 1≤i≤n1 \leq i \leq n (10 puntos)
  • Subtarea 2: n≤100n \leq 100, todos los elementos de aa son 11 o 22, es decir, ai∈{1,2}a_i \in \{1, 2\} para 1≤i≤n1 \leq i \leq n (17 puntos)
  • Subtarea 3: n≤100n \leq 100 (18 puntos)
  • Subtarea 4: n≤5000n \leq 5\,000 (15 puntos)
  • Subtarea 5: Sin restricciones adicionales (40 puntos)
